# Joker's Jackpot

Мини-казино на чистом HTML, CSS и JavaScript. Без сервера, без фреймворков и без реальных денег. Только виртуальные кредиты, неон, карты и желание нажать еще раз.

Играть можно тут:

https://vyacheslav666-cpu.github.io/joker-jackpot/

## Что внутри

- Переключаемые режимы колоды: 38, 54 и 108 карт.
- Стартовый баланс: 1000 кредитов.
- Ставки на Black, Red, Ace и Joker.
- История вытянутых карт справа от стола.
- Кнопки `+`, `-`, `x.5`, `x2` для быстрой настройки ставки.
- Кнопка `ДОДЕП +1000`, если хочется продолжить без драм.
- Кнопка `RESET`, если хочется начать с чистого стола.
- Анимации тасования, победы и проигрыша.
- Адаптивная верстка под телефон и desktop.

## Множители

| Ставка | Условие | Множитель |
| --- | --- | --- |
| Black | Пики или трефы | x1.9 |
| Red | Червы или бубны | x1.9 |
| Ace | Любой туз | x13 |
| Joker | Любой джокер | x26 |

## Режимы колоды

| Режим | Состав |
| --- | --- |
| 38 Cards | Карты от 6 до туза плюс 2 джокера |
| 54 Cards | Стандартная колода 52 карты плюс 2 джокера |
| 108 Cards | Две стандартные колоды плюс 4 джокера |

## Как играть

1. Выбери размер ставки.
2. Нажми на один из вариантов: `Black`, `Red`, `Ace`, `Joker`.
3. Карта вытянется сразу.
4. Если прогноз совпал, баланс пополнится на ставку с множителем.
5. Если не совпал, ставка сгорает.

## Локальный запуск

Просто открой `index.html` в браузере.

Если используешь VS Code, можно открыть через Live Server.

## Файлы

- `index.html` - разметка игры.
- `style.css` - внешний вид, адаптивность и анимации.
- `script.js` - колода, ставки, результат, выплаты и обновление интерфейса.

## Важно

Это учебная мини-игра. Тут нет платежей, вывода средств, аккаунтов и настоящего азартного функционала. Баланс существует только внутри страницы.
